03. GOGGLES : PARTICLE SYSTEM (HAIR)

Setting 1

PhysicsSystem 에서 Hair 선택
Emission 으로 Hair 빈도, 길이 조절

Setting 2

Children 설정으로 가서
Interpolated : 전체 면적에 고르게 털을 흩뿌려줌
Display Amount / Render Amount: 원본 털 1가닥당 생성할 자식 털의 개수(배수)
Clumping : 뭉침 효과
Clump (뭉침 강도) / Shape (형태 곡선)

Setting 3

털 가닥 자체의 꼬임, 헝클어짐, 잔털 같은 형태적 불규칙성
Random (무작위 잔털/부스스함) : 0.050로 끝부분이 아주 살짝 부스스하게 일어남
